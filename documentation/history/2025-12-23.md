## Key Decisions

- Cascade logic: marking yearly as done marks all quarters, months, weeks, days
- Period identifiers use formatted date strings (e.g., "2025-10-10", "2025-W01", "2025-Q4")
- Icon: lucide-style circle (not done) / check-circle (done)
